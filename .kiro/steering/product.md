# Product Overview

`@architect/utils` is a collection of common utility functions for the Architect framework ecosystem. It provides essential tools for AWS Lambda development, CLI operations, and infrastructure management.

## Core Purpose
- Standardized utilities shared across Architect projects
- AWS credential management and environment setup
- CLI progress indicators and status updates
- Path normalization and CloudFormation helpers
- Static asset fingerprinting for web applications

## Key Utilities
- **Banner**: Environment info display and AWS credential validation
- **Updater**: Standardized progress indicators and logging
- **Path/ID Conversion**: Cross-platform path handling and CloudFormation naming
- **Fingerprinting**: Static asset cache-busting for web apps
- **AWS Integration**: Lambda naming and credential checking

The package serves as foundational infrastructure for Architect's serverless development tools.
